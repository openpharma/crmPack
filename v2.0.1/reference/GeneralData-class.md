# `GeneralData`

**\[stable\]**

`GeneralData` is a class for general data input.

## Usage

``` r
.DefaultDataGeneral()
```

## Slots

- `ID`:

  (`integer`)  
  unique patient IDs.

- `cohort`:

  (`integer`)  
  the cohort (non-negative sorted) indices.

- `nObs`:

  (`integer`)  
  number of observations, a single value.

## Note

Typically, end users will not use the `.DefaultDataGeneral()` function.
