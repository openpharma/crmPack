# `NextBestThreePlusThree`

**\[stable\]**

`NextBestThreePlusThree` is the class for next best dose that implements
the classical 3+3 dose recommendation. No input is required, hence this
class has no slots.

## Usage

``` r
NextBestThreePlusThree()

.DefaultNextBestThreePlusThree()
```

## Note

Typically, end users will not use the `.DefaultNextBestThreePlusThree()`
function.

## Examples

``` r
# Next best dose class object using the classical 3+3 design.
my_next_best <- NextBestThreePlusThree()
```
