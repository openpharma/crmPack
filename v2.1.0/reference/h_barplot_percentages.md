# Convenience function to make barplots of percentages

Convenience function to make barplots of percentages

## Usage

``` r
h_barplot_percentages(x, description, xaxisround = 0)
```

## Arguments

- x:

  vector of samples

- description:

  xlab string

- xaxisround:

  rounding for xaxis labels (default: 0, i.e. integers will be used)

## Value

the ggplot2 object
