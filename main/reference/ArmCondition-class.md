# `ArmCondition`

**\[experimental\]**

`ArmCondition` is a virtual class for criteria controlling when a
[`DesignArm`](https://docs.crmpack.org/reference/DesignArm-class.md)
opens for enrollment in a
[`HierarchicalDesign`](https://docs.crmpack.org/reference/HierarchicalDesign-class.md).

## Usage

``` r
.DefaultArmCondition()
```

## Note

Typically, end users will not use the `.DefaultArmCondition()` function.
