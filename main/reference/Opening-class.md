# `Opening`

**\[experimental\]**

`Opening` is a virtual class for opening criteria, from which all other
specific opening criteria classes inherit. The subclasses are used for
backfill cohort designs.

## Usage

``` r
.DefaultOpening()
```

## Note

Typically, end users will not use the `.DefaultOpening()` function.
