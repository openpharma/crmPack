# Helper Function to Plot 2D Combination Evolution

**\[experimental\]**

Creates a 2D combination-plane plot with arrows showing step-to-step
dose transitions aggregated across simulations.

## Usage

``` r
h_plot_combo_evolution(sim_data)
```

## Arguments

- sim_data:

  (`list`) list of
  [`DataCombo`](https://docs.crmpack.org/reference/DataCombo-class.md)
  objects.

## Value

A `ggplot` object.
