# Helper Function to Create Doses Tried Plot

**\[stable\]**

Creates a bar plot showing average proportions of doses tested.

## Usage

``` r
h_plot_doses_tried(sim_doses, dose_grid)
```

## Arguments

- sim_doses:

  (`list`)  
  list of simulated doses per trial.

- dose_grid:

  (`numeric`)  
  dose grid.

## Value

A `ggplot` object.
