# `NoArmCondition`

**\[experimental\]**

`NoArmCondition` always opens an active arm immediately. It is the
default opening condition for
[`DesignArm`](https://docs.crmpack.org/reference/DesignArm-class.md)
objects.

## Usage

``` r
NoArmCondition()

.DefaultNoArmCondition()
```

## Note

Typically, end users will not use the `.DefaultNoArmCondition()`
function.
