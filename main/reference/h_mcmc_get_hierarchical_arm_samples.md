# Build Arm-Specific Sample Name Mappings for a `HierarchicalModel`

Build Arm-Specific Sample Name Mappings for a `HierarchicalModel`

## Usage

``` r
h_mcmc_get_hierarchical_arm_samples(model)
```

## Arguments

- model:

  (`HierarchicalModel`)\
  hierarchical model object.

## Value

Named list mapping arm-level sample names to hierarchical sample names.
