# Infer the Stochastic Node for a Parameter Reference

Infer the Stochastic Node for a Parameter Reference

## Usage

``` r
h_hierarchical_reference_stochastic_node(model, arm_name, ref)
```

## Arguments

- model:

  (`GeneralModel`)\
  arm-specific model object.

- arm_name:

  (`string`)\
  the user-facing arm name.

- ref:

  (`string`)\
  parameter reference.

## Value

Character scalar naming the stochastic node to pool.
