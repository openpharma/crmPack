# Show `HierarchicalData` Objects

**\[experimental\]**

Display a brief representation of the
[`HierarchicalData`](https://docs.crmpack.org/reference/HierarchicalData-class.md)
object.

## Usage

``` r
# S4 method for class 'HierarchicalData'
show(object)
```

## Arguments

- object:

  (`HierarchicalData`)\
  the object we want to print.

## Value

Invisibly returns the object itself.
