# Find Stochastic Subexpressions in an Expression

Find Stochastic Subexpressions in an Expression

## Usage

``` r
h_hierarchical_stochastic_subexpressions(expr, stochastic_roots)
```

## Arguments

- expr:

  (`language`)\
  expression to inspect.

- stochastic_roots:

  (`character`)\
  stochastic node roots.

## Value

Named list of stochastic subexpressions keyed by expression text.
