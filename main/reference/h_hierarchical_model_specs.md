# Suffix Single-Arm Model Specifications

Suffix Single-Arm Model Specifications

## Usage

``` r
h_hierarchical_model_specs(model, arm_name, from_prior)
```

## Arguments

- model:

  (`GeneralModel`)\
  arm-specific model object.

- arm_name:

  (`string`)\
  hierarchical arm name.

- from_prior:

  (`flag`)\
  whether to request prior-only specifications.

## Value

A named list of suffixed model specifications.
