# Is a Compatible Hierarchical Single-Agent Model

**\[experimental\]**

Uses the same structural compatibility contract as
[`TwoDrugsCombo()`](https://docs.crmpack.org/reference/TwoDrugsCombo-class.md)
for single-agent binary outcome models.

## Usage

``` r
h_hierarchical_is_single_model(model)
```

## Arguments

- model:

  (`GeneralModel`)\
  model object.

## Value

A flag.
