# `NextBest`

**\[stable\]**

`NextBest` is a virtual class for finding next best dose, from which all
other specific next best dose classes inherit.

## Usage

``` r
.DefaultNextBest()
```

## Note

Typically, end users will not use the `DefaultNextBest()` function.
