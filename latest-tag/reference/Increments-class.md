# `Increments`

**\[stable\]**

`Increments` is a virtual class for controlling increments, from which
all other specific increments classes inherit.

## Usage

``` r
.DefaultIncrements()
```

## Note

Typically, end users will not use the `.DefaultIncrements()` function.
