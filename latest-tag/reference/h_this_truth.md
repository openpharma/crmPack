# Helper Function to call truth calculation

Helper Function to call truth calculation

## Usage

``` r
h_this_truth(dose, this_args, truth)
```

## Arguments

- dose:

  (`number`)  
  current dose.

- this_args:

  (`data.frame`)  
  list of arguments for the truth.

- truth:

  (`function`)  
  defines the true probability for a DLT at a dose.

## Value

The updated `this_truth`.
